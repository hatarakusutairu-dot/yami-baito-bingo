# 🚨 闇バイトビンゴ

**ゲームで学ぶ闇バイトの危険ワード — 知ることが、自分を守る第一歩**

高校生向けの授業で使える、闇バイトの危険ワードを楽しく学べる教育用Webアプリです。

## 🎮 3つのモード

### 🎯 ビンゴモード
- 「次のワード」を引いてボード上のワードをタップ
- 縦・横・斜めが揃ったらビンゴ＝犯罪者！
- クラス全体でプロジェクター投影しながら使用可能

### 🧠 神経衰弱モード
- 危険ワードとその意味のペアを見つけるマッチングゲーム
- 6 / 8 / 10ペアの難易度選択
- 個人・ペア学習に最適

### 📚 危険ワード一覧
- 全30語をカテゴリ別に確認（隠語・甘い言葉・手口）
- 危険度レベル表示
- 授業の解説・振り返りに

## 📋 収録ワード例

| カテゴリ | ワード例 |
|---------|---------|
| 🔐 隠語 | UD、叩き（タタキ）、かけ子、受け子、出し子、手押し、炊飯器、SIM案件 |
| 🍬 甘い言葉 | 高額報酬、即日即金、ホワイト案件、リスクなし、誰でも簡単、〜するだけ |
| ⚠️ 手口 | テレグラム、シグナル、身分証送って、家族の情報教えて、DMで連絡 |

## 🚀 セットアップ

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
